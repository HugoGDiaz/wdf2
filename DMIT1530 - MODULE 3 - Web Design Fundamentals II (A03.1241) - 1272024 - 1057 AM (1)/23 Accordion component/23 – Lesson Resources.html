ARIA Authoring Practices Guide (APG): Accordion Pattern (Sections With Show/Hide Functionality) 

https://www.w3.org/WAI/ARIA/apg/patterns/accordion/

An accordion is a vertically stacked set of interactive headings that each contain a title, content snippet, or thumbnail representing a section of content.

ARIA Authoring Practices Guide (APG): Accordion Example

https://www.w3.org/WAI/ARIA/apg/patterns/accordion/examples/accordion/

gov.uk Design System

https://design-system.service.gov.uk/components/accordion/

This is part of the British government's design system. On this page, we can see all of the rules they have for using accordion components, including when and why to use them and accessibility concerns. This is a great example because government websites are information-dense and need to be accessible to the entire population.
